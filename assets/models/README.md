# Modelos 3D

Esta carpeta contiene los modelos 3D en formato GLB para la galería.

## Cómo agregar modelos

Para usar esta galería con tus propios modelos 3D:

1. Coloca tus archivos `.glb` en esta carpeta
2. Asegúrate de que los nombres coincidan con los que se usan en `index.html`
3. Ejemplos:
   - `part_01.glb` - Pieza Industrial A
   - `part_02.glb` - Pieza Industrial B

## Recursos gratuitos de modelos 3D

Puedes encontrar modelos 3D gratuitos en los siguientes sitios:

- [Sketchfab](https://sketchfab.com/search?features=downloadable&type=models)
- [Google Poly](https://poly.google.com/)
- [Archive 3D](https://archive3d.net/)
- [GrabCAD](https://grabcad.com/library)

## Optimización de modelos

Para mejorar el rendimiento:

- Mantén el tamaño del archivo por debajo de 10 MB
- Usa compresión Draco durante la exportación
- Reduce la cantidad de triángulos innecesarios
- Combina texturas en un atlas UV cuando sea posible
